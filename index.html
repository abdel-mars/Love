Oh Branches,

I can always use git branch to know which one I'm in.

Or, to start anew, git branch -b [new name] while create a new branch,

Perhaps one to attach my tree swing to.


To merge two branches into one, it's as simple as running through a field of flowers,

just by typing git merge [branch name].

And lest I forget,

the power of git remote.


Because git remote -v will let me see my options,

and fetch will take from the server to merge.

Even though your love pushes and pulls me around,

the only place I want to checkout is with you.


Tyler :)
